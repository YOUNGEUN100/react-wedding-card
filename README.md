## 설치 및 실행 방법
### 프로젝트 클론
```bash
git clone https://github.com/YOUNGEUN100/react-wedding-card.git
```
### 프로젝트 디렉토리로 이동
```bash
cd repository
```
### 의존성 설치
```bash
npm install
```
### 서버 실행
```bash
npm start
```
